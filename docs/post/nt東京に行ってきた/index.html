NT東京

今日科学技術館に開催したNT東京に行ってきた。

江戸城、参ります！

やっと着いた。

会場に入った後目に映るのはロボットバンド。

自作音源とMIDIキーボードもありそう。

C#で作ったアンドロイドとハードウェアの通信アプリ。C#でアンドロイドアプリの開発もできることは初めて知ってた。

深層学習に基づくカメラ認識のロボット操作。

これはハードディスクドライブかな。日本語を上手く喋らないのでどのために作ったのは聞かなかった、ちょっと残念。でも面白そう。

やばいものを作ったらしいｗ。

ARで現実に影響を与えられる球形ロボット。
