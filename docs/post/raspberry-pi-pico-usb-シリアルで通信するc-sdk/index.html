Raspberry Pi Pico - USB シリアルで通信する（C SDK）

Raspberry Pi Pico の USB よりシリアル通信を実現した。

2月 01, 2025
読了時間: 1分

CMakeLists.txt

以下のコードを追加してください。

1
2

# serial port
pico_enable_stdio_usb(project_name 1) # stdio_usb を有効化する

初期化

最初で stdio.h をインクルードしてください。

1

#include "stdio.h"

そしてプロジェクトのメイン関数で以下のコードを追加してください。

1

stdio_init_all(); // stdio_usb サポートを有効化する
